We're in the year 2014: Too bad <blink> tags do not work in modern browsers any more...

Using Web Components, you can get it back: <x-blink>

 

How to?

- Get x-blink.html from this repository and put it in your directory
- Add the following lines to the head of your document:

    <script src="http://cdnjs.cloudflare.com/ajax/libs/polymer/0.3.4/platform.js"></script>
    <script src="http://cdnjs.cloudflare.com/ajax/libs/polymer/0.3.4/polymer.js"></script>
    <link rel="import" href="x-blink.html">

- Use <x-blink>Wow, so amazing!</x-blink> to make the text blink. No more JavaScript needed. Wow, so amazing!
